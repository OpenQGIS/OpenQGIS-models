# OpenQGIS-models
QGIS常用模型分享
## 1. Joy-Plot​
[叠嶂图生成模型 下载](https://github.com/OpenQGIS/OpenQGIS-models/blob/main/model/Joy-Plot%E2%80%8B.model3)
> **DEM** -- 输入的栅格图层
> 
> **CRS** -- 生成的网格坐标系（必须为投影坐标）
> 
> **X-Distance（km）**-- 横向山脊线转折点的间距（千米）
> 
> **Y-Distance（km）**-- 纵向每一个山脊线的间距（千米）
> 
> **Scale** -- 夸张系数，数值越大越立体
>
> **BottomPoint_height** -- 基座高度（千米）
